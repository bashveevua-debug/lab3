### Логика работы

**Блокировка пользователей**:
	- Каждый пользователь имеет свой список заблокированных (хранится в `blocked_users_table`)
	- Когда пользователь A отправляет сообщение пользователю B, проверяется, не заблокировал ли B пользователя A
	- Если заблокировал - сообщение не доставляется

**Цензура сообщений**:
   - Система поддерживает динамический список запрещённых слов (`censored_words_table`)
   - Все сообщения проходят цензуру: запрещённые слова заменяются звёздочками


### Запуск
поднимаем контейнеры:
docker compose up -d

### Тестирование

далее прокидваем список плохих слов

это можно сделать через kafka ui в браузере http://127.0.0.1:8080/

в топик censored_words пишем:
```
{"word": "дура", "action": "add"}
{"word": "фигня", "action": "add"}
```

далее Сабина блокирует Свету

в топик blocked_users пишем:
```
{
    "user": "sabina",
    "blocked_user": "sveta",
    "action": "block"
}
```

далее пресылаем сообщения между пользователями

в топик messages пишем:
```
{
    "sender": "sabina",
    "receiver": "sveta",
    "text": "Привет! Как дела по таске ?"
}

{
    "sender": "sveta",
    "receiver": "sabina",
    "text": "Дура ! как я тебе отвечу ? ты меня забанила ?"
}

{
    "sender": "sabina",
    "receiver": "lera",
    "text": "Лера ! Узнай у Светы как дела по таске , эта дура мне не отвечает !"
}
```

{
